Interfaces Gráficas no Linux

O Que são?

As interfaces gráficas no Linux são UX/UI responsáveis pela interação do usuário com o sistema de forma gráfica.

As principais

  • Gnome
  • KDE Plasma
  • XFCE
  • Cinnamon
  • MATE
  • Budgie

Suas diferenças

Dentro desse universo existem vários frameworks gráficos no mundo Linux dentre eles os dois mais famosos são o GTK e o QT, sendo o GTK mantido pela equipe do Gnome e o QT mantido pela equipe do KDE, a maioria das interfaces menores como XFCE, MATE entre outras escolhem um desses dois frameworks gráficos para gerir sua UX/UI.
